Encabezados

Los encabezados en un documento HTML se utilizan para indicar diferentes niveles de títulos o secciones dentro de una página web.

Párrafos y saltos de línea

Los párrafos definen bloques de texto en un documento HTML. El texto delimitado entre la etiqueta inicial <p> y la etiqueta final </p> se considera un párrafo.

Los saltos de línea permiten que el texto de un documento HTML comience en una nueva línea, sin necesidad de comenzar un nuevo párrafo.

Negritas, cursivas y subrayado

La etiqueta <b> permite visualizar en negritas el texto colocado entre la etiqueta inicial y la etiqueta final </b>.

La etiqueta <i> permite visualizar en cursivas el texto colocado entre la etiqueta inicial y la etiqueta final </i>.

La etiqueta <u> permite visualizar subrayado el texto colocado entre la etiqueta inicial y la etiqueta final </u>.
